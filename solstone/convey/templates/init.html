sol agent

the sol agent curates your journal and is powered by ai. you can skip this and set it up later in settings, either a hosted provider or a local model via ollama.

a hosted provider key is the recommended way to start. running a capable model locally is a real, supported goal, but it needs capable hardware and a model with strong "thinking" support. it isn't the default daily experience yet. how to power sol →

the fastest start is a gemini key; enter it below. note this is a developer api key from Google AI Studio (aistudio.google.com) — not a gemini.google.com chat account. you can tune how much automated processing runs in settings.

if you're a scout for solstone you can enter the key provided here, or apply to get a key for testing at scouts.solstone.app

prefer Claude or OpenAI? add a developer key in settings → API keys: Claude from console.anthropic.com (not the claude.ai chat signup), OpenAI from platform.openai.com (not chatgpt.com).
